# Looker-Dashboards-_Systems


Project Dashboard Images

-------------------------------------------------------------------
This repository stores visualization snapshots of various Project Management and Looker Dashboards. These images provide an overview of key performance indicators (KPIs), progress, and operational data related to the project.
-----------------------------------------------------------------
--Purpose---
The primary objective of this repository is to document and share the latest views of all relevant dashboards in a centralized location. This helps stakeholders review the status without requiring direct access to the live dashboards.
--------------------------------------------------------------
--Dashboard Overview-----
The following images are currently available in the repository, listed by their file names:
File Name	Description
-Attendance.png =	Team attendance and time tracking metrics.
-Home Page Dashboard.png =	This is to display the correct dashboard pages in one place..
-IMS Dashboard.png =	Inventory Management System (IMS) status and stock levels.
-Material Received Dashboard.png	= Details of received materials and supply chain data.
-Material Requirement Dashboard.png =	Analysis of upcoming needs and material planning.
-PMS Dashboard.png	= Performance/Project Management System tracking.
-Site Task Report.png	= Report of ongoing tasks, status, and timelines at the site.
